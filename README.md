# ray-tracing-in-one-weekend-series-study
A physically based path tracer implemented in C++ from scratch. Based on Peter Shirley's "Ray Tracing in One Weekend Series".

This project is based on the book [_Ray Tracing in One Weekend_](https://raytracing.github.io/books/RayTracingInOneWeekend.html), [_Ray Tracing: The Next Week_](https://raytracing.github.io/books/RayTracingTheNextWeek.html), and [_Ray Tracing: The Rest of Your Life_](https://raytracing.github.io/books/RayTracingTheRestOfYourLife.html).
